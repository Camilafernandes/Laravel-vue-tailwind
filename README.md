## Requisitos técnicos

* PHP 7^
* Mysql 5.6
* Composer

## Após o clone do projeto

Execute os seguintes comandos:

`composer install`

`cp .env.example .env`

`npm run production`

## Rodar aplicação

```bash
php artisan serve
```

## URLS

```bash
    localhost:8000 - Cadastro
    localhost:8000/users/edit/1  - Edição
    localhost:8000/users/edit/1  - Edição
    localhos:8000/slideshow - Slideshow
```
